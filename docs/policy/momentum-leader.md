# 强势领涨策略

## 一句话描述

近 5 日涨幅位于市场前 10%，且当日继续放量上涨。

## 适用周期与风格

- 周期：短线（1-5 个交易日）
- 风格：动量 / 趋势延续

## 数据源与关键字段

- 数据源：本地 `StockDailyQuote` 表
- 关键字段：
  - `stock_code`：股票代码
  - `trade_date`：交易日期
  - `close`：收盘价
  - `change_pct`：涨跌幅
  - `amount`：成交额

## 选股逻辑

1. 获取最近 20 个日历日内的 `StockDailyQuote` 数据
2. 取最近 5 个交易日数据
3. 计算每只股票近 5 日累计涨幅：
   - `return_5d = (第 5 日收盘价 - 第 1 日收盘价) / 第 1 日收盘价`
4. 取涨幅前 10% 的股票
5. 过滤当日继续上涨且成交额大于 5 日均成交额
6. 应用公共过滤：ST、科创板、创业板、北交所、次新股
7. 计算原始评分：
   - `rank_pct`：5 日涨幅排名百分位（前 10% 对应 90-100）
   - `volume_ratio = 当日成交额 / 5 日均成交额`
   - `score = rank_pct * 0.4 + change_pct * 0.3 + volume_ratio * 0.3`
8. 归一化到 0-100 后取 Top10

## 推荐理由模板

```
近 5 日涨幅 {return_5d:.2f}%，位于市场前 {rank_pct:.0f}%，当日继续上涨 {change_pct:.2f}%
```

## 输出字段

| 字段 | 说明 |
|---|---|
| `code` | 股票代码 |
| `name` | 股票名称 |
| `score` | 归一化评分（0-100） |
| `buy_price` | 当日收盘价（参考买入价） |
| `reason` | 推荐理由 |

## 风险与注意事项

- 动量策略在震荡市或反转节点容易出现追高被套
- 依赖本地 `StockDailyQuote` 表，需确保已同步最近 5 个交易日数据
- 当日放量上涨也可能是短期出货信号，需结合板块和大盘判断
