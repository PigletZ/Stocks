# 首板不破·龙头跟踪策略（piglet）

## 一句话描述

个人摸索用策略：捕捉近期首次涨停后、股价持续不跌破首板的强势龙头股，并标注是否有龙虎榜游资介入。持续完善中。

## 适用周期与风格

- 周期：最近 5 个交易日
- 风格：龙头 / 强势股

## 数据源与关键字段

- **涨停板名单**：Tushare `limit_list_d`（构建涨停池、确定首板日）
  - 关键字段：`trade_date`、`ts_code`、`limit`（U=涨停 / Z=炸板 / D=跌停）
- **龙虎榜**：Tushare `top_list`（增强标记，不参与扩池/筛除）
  - 关键字段：`trade_date`、`ts_code`、`net_amount`（龙虎榜净买额）
- **个股日行情**：本地 `StockDailyQuote` 表
  - 关键字段：`close`（首板日基准价、当日收盘）、`low`（判断是否不破）、`amount`（成交额，复盘展示）

## 选股逻辑

1. 取最近 5 个交易日的涨停板名单（`limit_list_d` 中 `limit = U`），按个股去重，构建**涨停候选池**。
2. 应用公共过滤：ST、科创板（688）、创业板（300/301）、北交所（8/43/82/83/87/88/89/920）、次新股（上市不满 365 天）。
3. 对候选池中每只股票，确定其在 5 日窗口内的**首次涨停日**（first_limit_date），并取该日**收盘价**作为基准价 `base_close`。
4. **核心条件——首板不破**：从首次涨停日的**次日**起，到最近交易日为止，每个交易日的**最低价 `low` ≥ `base_close`**，则视为"持续不破"，纳入结果。
   - 若首次涨停日就是最近交易日（无后续交易日可验证），**当日不纳入**，等次日数据更新后自然判定。
5. **龙虎榜增强标记（方案B）**：对纳入结果的个股，检查其在 5 日窗口内是否上过龙虎榜（`top_list`）。
   - 上榜 → 标记 `游资介入 = 是`，并记录窗口内龙虎榜**净买额合计** `net_amount`。
   - 未上榜 → `游资介入 = 否`。
   - 注意：龙虎榜**仅用于标记，不增减选股池**，是否上榜都不影响是否入选。

## 输出字段

- 暂不评分，推荐理由、买入价留空。
- 列表展示个股信息，并附带：
  - `首板日期`（first_limit_date）
  - `游资介入`（是/否）
  - `龙虎榜净买额`（窗口内合计，未上榜为空）
- 排序建议：按 `首板日期` 升序（越早启动、不破天数越多 → 越强）；游资介入者可置顶或高亮。
- 点击个股时，展示其最近 5 个交易日的基础信息（每日 `close` / `low` / `change_pct` / `amount`，并标出首板日与当日是否跌破 `base_close`）。

## 待完善 / 备忘

- 5 日窗口可参数化，便于回测调优。
- 是否加入成交量确认（如不破期间是否缩量）待观察后决定。
