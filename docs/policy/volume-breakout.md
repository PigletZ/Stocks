# 放量突破策略

## 一句话描述

当日成交量为 20 日均量 2 倍以上，且收盘价创 20 日新高。

## 适用周期与风格

- 周期：短线至波段（3-10 个交易日）
- 风格：量价突破 / 趋势启动

## 数据源与关键字段

- 主要数据源：本地 `StockDailyQuote` 表
- 关键字段：
  - `stock_code`：股票代码
  - `trade_date`：交易日期
  - `close`：收盘价
  - `amount`：成交额

## 选股逻辑

1. 获取最近 45 个日历日内的 `StockDailyQuote` 数据
2. 取最近 20 个交易日
3. 对每只股票计算：
   - 20 日平均成交额 `amount_ma20`
   - 20 日最高价 `high_20`
4. 过滤条件：
   - 当日成交额 >= 20 日均额 * 2
   - 当日收盘价 >= 20 日最高价
5. 应用公共过滤：ST、科创板、创业板、北交所、次新股
6. 计算原始评分：
   - `volume_ratio = 当日成交额 / 20 日均额`
   - `breakout_pct = (收盘价 - 20 日最高) / 20 日最高`
   - `score = volume_ratio * 50 + breakout_pct * 100 * 50`
7. 归一化到 0-100 后取 Top10

## 推荐理由模板

```
成交额是 20 日均额 {volume_ratio:.1f} 倍，创 20 日新高
```

## 输出字段

| 字段 | 说明 |
|---|---|
| `code` | 股票代码 |
| `name` | 股票名称 |
| `score` | 归一化评分（0-100） |
| `buy_price` | 当日收盘价（参考买入价） |
| `reason` | 推荐理由 |

## 风险与注意事项

- 依赖本地 `Bar` 表数据，若未同步完整可能导致结果为空
- 放量突破后可能存在假突破风险，建议结合市场环境判断
- 对成交量异常放大（如解禁、复牌）需人工甄别
