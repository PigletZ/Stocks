# 龙虎榜游资追踪策略

## 一句话描述

基于当日龙虎榜数据，跟随游资净买入，聚焦主板小市值标的。

## 适用周期与风格

- 周期：短线（1-5 个交易日）
- 风格：事件驱动 / 游资跟随

## 数据源与关键字段

- 数据源：Tushare `top_list`
- 关键字段：
  - `ts_code` / `name`：股票代码与名称
  - `close`：收盘价
  - `pct_change`：涨跌幅
  - `net_amount`：龙虎榜净买额
  - `net_rate`：净买额占成交额比例
  - `amount`：成交额
  - `float_values`：流通市值（元）

## 选股逻辑

1. 获取当日龙虎榜数据
2. 过滤 `net_amount > 0`（游资净买入为正）
3. 应用公共过滤：ST、科创板、创业板、北交所、次新股
4. 流通市值过滤：`float_values <= 200 亿`
5. 计算原始评分：
   - `score = net_rate * 0.4 + pct_change * 0.3 + log(amount + 1) * 0.3`
6. 归一化到 0-100 后取 Top10

## 推荐理由模板

```
龙虎榜净买占比 {net_rate:.2f}%，主板小市值（流通市值 {float_mv/1e8:.1f} 亿）
```

## 输出字段

| 字段 | 说明 |
|---|---|
| `code` | 股票代码 |
| `name` | 股票名称 |
| `score` | 归一化评分（0-100） |
| `buy_price` | 当日收盘价（参考买入价） |
| `reason` | 推荐理由 |

## 风险与注意事项

- 龙虎榜数据次日开盘前可用，适合盘后选股
- 游资席位可能存在诱多风险，需结合其他维度判断
- 流通市值字段为空时不做过滤，避免漏选
