# 概览页

## 简介
概览页主要包含：实时概览，日概览，涨停，跌停，龙虎榜，以tag页的方式展示

## Tag页功能
### 日概览
- 支持选择日期，展示指定日期的日概览信息，默认展示当天的日概览信息，若当天未开盘或不是交易日，则展示上一个交易日的日概览信息
- 日概览展示内容
    - 主要指数板块
        - 包含以下指数信息
            - 上证指数
            - 深证成指
            - 创业板指
            - 科创50
            - 北证50
        - 鼠标悬浮时，显示其最近30日K线图
    
    - 市场统计板块
        - 包含以下卡片信息
            - 涨跌家数
            - 平盘家数
            - 涨停
            - 跌停
            - 炸板
            - 总成交额

    - 市场情绪趋势
        - 形式为折线图，可选数据有涨跌家数，涨停数，跌停数
        - 支持选择近一周，近一个月，近三月以及指定起始时间
        - 时间轴以顺序方式

    - 板块排行
        - 行业板块涨幅
            - 涨跌幅都只统计前50个
            - 支持按涨跌幅排序
            - 支持按成交额排序
        - 概念板块涨幅
            - 涨跌幅都只统计前50个
            - 支持按涨跌幅排序
            - 支持按成交额排序
- 数据同步
    - 支持手动同步当天的数据
- 数据来源
    - 采集：数据在每个交易日下午4点收盘后，统计好进行入库
    - 页面展示的数据，都是从数据库读取

### 涨停
- 支持选择日期，展示指定日期的涨停信息
- 默认展示当天涨停的个股信息
    - 个股信息包含
        - 股票代码
        - 股票名称
        - 最新价
        - 涨跌幅
        - 成交额
        - 封单比
        - 换手率
        - 流通值
        - 总市值
        - 首封时间
        - 封住时间
        - 开板次数
        - 几天几板
        - 涨停分析
    - 操作
        - 复盘跳转
        - 加入自选
- 数据需入库，后续查询默认查库


### 跌停
- 支持选择日期，展示指定日期的跌停信息
- 默认展示当天跌停的个股信息
    - 个股信息包含
        - 股票代码
        - 股票名称
        - 最新价
        - 涨跌幅
        - 成交额
        - 封单比
        - 换手率
        - 流通值
        - 总市值
        - 首封时间
        - 封住时间
        - 开板次数
        - 跌停分析
- 数据需入库，后续查询默认查库

### 龙虎榜
- 支持按日期筛选
- 龙虎榜数据
    - 游资
        - 展示代码、名称、上榜日、收盘价、涨跌幅、龙虎榜净买额、上榜原因
    - 机构
        - 展示代码、名称、上榜日、收盘价、涨跌幅、龙虎榜净买额、上榜原因
- 支持复盘跳转、加入自选
